# cyart-vapt-team

This repository contains weekly documentation and deliverables for
Vulnerability Assessment and Penetration Testing (VAPT) labs.

## Structure
- Week 2: Reconnaissance, Exploitation, Post-Exploitation, Capstone VAPT
- Week 3:Advanced Exploitation Lab,Web Application Testing Lab,Reporting Practice,Post-Exploitation and Evidence Collection,Capstone Project: Full VAPT Cycle.

## Disclaimer
All testing was performed on intentionally vulnerable lab environments
for educational purposes only.
